Each python `main.py` can be started in IDE terminal or by running a command:
*python3 main.py*


**branch deadline_calculator**
simple python program calculated an amount of hours remaining until specific deadline. Program works with datetime library and creates datetime objects to calculate hours remaining. 

**branch spreadsheet**
- program uses openpyxl library 
- calculates number of products per supplier
- total inventory value per supplier
- identifies products having inventory less than 10
- adding a new column number 5 and saves the files with a new name

**branch oop**
- `post.py` contains definition of Post class
- `user.py` contains definition of User class
- `main.py` creates instances of Post and User class and calls their methods. 

**branch gitlab_API**
- using `requests` python library all my projects available on my GitLab repository are returned as name and URL
